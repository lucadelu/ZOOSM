How to upload data

To upload data on OpenStreetMap database, you should login with the "Login" button and then activate selection with "Select features", with this it's possible to select more features to upload. To unselect the feature/s there are several way:
  • to unselect all features it's possible clicking "Unselect features" or clicking in the map where there aren't features
  • to unselect one feature you should click the feature that you want unselect
At the end when you chosen the features that you want upload you press "Upload" button
